# 🚀 快速部署指南（无需注册）

## 方法1：Netlify Drop（最简单，30秒完成）

1. **确保项目已构建**
   ```bash
   npm run build
   ```
   这会在 `dist/` 文件夹中生成所有文件

2. **打开 Netlify Drop**
   - 访问：https://app.netlify.com/drop
   - 无需注册，无需登录

3. **拖放部署**
   - 打开文件管理器，找到项目的 `dist` 文件夹
   - 将整个 `dist` 文件夹拖到 Netlify Drop 页面上
   - 等待上传完成（通常10-30秒）

4. **完成！**
   - 你会立即得到一个 URL，例如：`https://random-name-123456.netlify.app`
   - 这个 URL 可以立即分享给任何人

5. **（可选）自定义域名**
   - 如果你想要更好记的名字，可以点击 "Change site name"
   - 例如改成：`my-snake-game.netlify.app`

---

## 方法2：使用 Vercel（需要注册，但功能更强大）

### 注册 Vercel 账号：

1. **访问 Vercel**
   - 打开：https://vercel.com/signup
   
2. **选择注册方式**（推荐使用 GitHub）
   - 点击 "Continue with GitHub"
   - 如果没有 GitHub 账号，可以用邮箱注册

3. **完成注册**
   - 授权 Vercel 访问
   - 填写基本信息

4. **回到命令行部署**
   ```bash
   vercel login
   vercel --prod
   ```

## ❓ 常见问题

**Q: Netlify Drop 的链接会过期吗？**
A: 不会！链接永久有效，除非你手动删除。

**Q: 我可以更新已部署的网站吗？**
A: Netlify Drop 每次都是新部署。如果需要更新，建议注册 Vercel 或 Netlify 账号。

**Q: 部署后可以看到访问统计吗？**
A: Netlify Drop 不提供统计。如需统计，建议注册账号。
